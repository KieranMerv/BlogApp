Welcome to PrimaryBlog

A simple blog, for anyone.

This is a minimalist, simple blog application for anyone to pen down any thoughts, emotions, experiences, expressions, secrets, anything that their hearts desire. It is a safe place for anybody. It can be shared publicly on the front page, or it can be private, i.e. only accessible to the author.

This app is a SPA built using Angular and .Net Core, and features basic CRUD operations and authentication powered by Microsoft Identity. It serves as part of the portfolio of Mervin Wee, an aspiring web developer in the midst of a mid-career switch.

While care has been taken to make this application secure, it has not undergone industrial testing. It is therefore not recommended for any sensitive information to be input as credentials. Register with a fake email address, or an email address that is just for casual use.

Thank you, and please enjoy this app!
